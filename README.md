## weex-vue-router页面跳转

### 下载

```
git clone https://github.com/heuuLZP/weex-vue-router.git
```

### web端

编译

```
npm run build
```

启动服务

```
npm run serve
```

监听变化

```
npm run dev
```



### 文件目录

```
src
├── APP.vue               ## 根组件
├── main.js               ## 入口文件
├── page
│   ├── AboutView.vue     ## about页
│   └── HomeView.vue      ## home页
└── router.js             ## 路由
```
